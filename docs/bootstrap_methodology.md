# ブートストラップ検定の手法概要と理論的根拠

本ドキュメントでは、本プロジェクトにおけるABテストシミュレーションで採用している **クラスター・ブートストラップ検定（Cluster Bootstrap Test）** の手法概要、統計学的な課題背景、および数理的な理論根拠について解説します。

---

## 1. 課題背景：セッション相関とZ検定の限界

### 1.1 独立同一分布 (i.i.d.) の仮定の崩壊
通常の「2標本の母比率の差の検定（z検定）」は、すべての観測値（セッション単位のコンバージョン有無 $Y_{ij} \in \{0, 1\}$）が互いに独立かつ同一の分布（independent and identically distributed: **i.i.d.**）に従うことを前提としています。

しかし、同一ユーザー $i$ が複数回セッション $j$ を行う実務のABテストでは、以下の要因によってセッション間の独立性が失われます。
- **ユーザー異質性 (User Heterogeneity)**: ユーザーごとに基本コンバージョン率が異なる（例：熱心なユーザーは元々CVしやすく、ライトユーザーはCVしにくい）。
- **ユーザー内相関**: 同一ユーザー内の行動（セッション）は、他人のセッションよりも互いに高い正の相関を持ちます。

### 1.2 標準誤差の過小評価と第一種過誤のインフレ
セッション間の相関を無視して通常のz検定を適用すると、統計学的に以下の問題が発生します。

1. **実質的な情報量の過大評価**: 
   相関のある5つのセッションは、独立した5つのセッションほどの情報量を持ちません。しかし、z検定は全セッション数 $N = \sum n_i$ をそのまま分母（サンプルサイズ）として処理します。
2. **標準誤差 (Standard Error: SE) の過小評価**: 
   比率の差のばらつきを示す理論上の標準誤差 $SE$ が、真のばらつきよりも極端に小さく計算されます。
3. **p値の過小評価と第一種過誤 (Type I Error) のインフレ**: 
   標準誤差が小さく見積もられることで、統計量 $z = \frac{\hat{p}_B - \hat{p}_A}{SE}$ が不当に大きくなり、p値が有意水準（$\alpha = 0.05$）を下回りやすくなります。結果として、「実際には両群に差がない（帰無仮説が真）」にもかかわらず、誤って「有意差あり」と判定してしまう確率（第一種過誤率）が $5\%$ を大幅に超えてインフレします（過剰棄却）。

---

## 2. クラスター・ブートストラップの手法概要

この相関構造によるバイアスを排除するために、本プロジェクトではセッション単位ではなく、**ユーザー単位（クラスター単位）でリサンプリングを行う「クラスター・ブートストラップ」**を採用しています。

### 2.1 アルゴリズムの手順

元のデータセットにおいて、A群のユーザー数を $M_A$、B群のユーザー数を $M_B$ とします。ブートストラップ反復回数を $B$ 回（デフォルトでは $1000$ 回）とします。

各反復 $b = 1, 2, \dots, B$ について、以下の処理を行います。

1. **ユーザー単位の復元抽出**:
   - A群のユーザー集合 $\{U_{A, 1}, \dots, U_{A, M_A}\}$ から、重複を許して $M_A$ 個のユーザーをランダムにサンプリングします。
   - B群のユーザー集合 $\{U_{B, 1}, \dots, U_{B, M_B}\}$ からも同様に、重複を許して $M_B$ 個のユーザーをランダムにサンプリングします。
2. **データの結合（クラスターの維持）**:
   - サンプリングされたユーザーに対応するすべてのセッションログを丸ごと複製・結合し、疑似的な標本データ（Bootstrap Sample）を作成します。これにより、**ユーザー内部のセッション数やCV率の相関構造が完全に維持**されます。
3. **疑似統計量の算出**:
   - 疑似標本におけるA群・B群それぞれの総コンバージョン率 $\hat{p}^*_{A, b}$ および $\hat{p}^*_{B, b}$ を計算し、その差を記録します。
     $$\theta^*_b = \hat{p}^*_{B, b} - \hat{p}^*_{A, b}$$

---

## 3. 二標本ブートストラップ仮説検定の理論的根拠

得られたブートストラップ統計量の分布 $\{\theta^*_1, \dots, \theta^*_B\}$ を用いて、帰無仮説 $H_0: \theta = 0$（真の差はゼロである）に対する検定を行います。

### 3.1 帰無仮説を満たすための中心化（シフト）
ブートストラップでサンプリングされた差の平均は、元のサンプルにおける実測の差 $\theta_{\text{obs}} = \hat{p}_B - \hat{p}_A$ に近似します。
仮説検定の定義（帰無仮説 $H_0$ が正しいという条件下での確率計算）を満たすため、分布全体を実測値分だけ左または右に平行移動（中心化）させます。

$$\theta^*_{\text{shifted}, b} = \theta^*_b - \theta_{\text{obs}}$$

この変換により、$\{\theta^*_{\text{shifted}, 1}, \dots, \theta^*_{\text{shifted}, B}\}$ は**「平均がぴったり $0$（差がない世界）」でありながら、「相関関係を維持した状態での比率の差の真のばらつき（標準誤差）」を正確に表現したサンプリング分布**になります。

### 3.2 両側検定p値の算出
この中心化された分布において、実際の観測値である $\theta_{\text{obs}}$ と「同等以上に極端な値」が発生する確率をp値として算出します。

$$p = \frac{1}{B} \sum_{b=1}^{B} I\left( \left| \theta^*_{\text{shifted}, b} \right| \ge \left| \theta_{\text{obs}} \right| \right)$$

ここで $I(\cdot)$ は指示関数であり、条件が真のとき $1$、偽のとき $0$ を返します。

もし、このp値が事前に設定した有意水準 $\alpha$（例: $0.05$）を下回った場合、「差がないという仮定（H0）の下では、実測された差 $\theta_{\text{obs}}$ は偶然では滅多に起こりえない極めて稀なイベントである」と解釈し、帰無仮説を棄却して「統計的に有意な差がある」と判断します。
